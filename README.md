# Social Media Engagement Dashboard

## Objective

Analyze social media engagement metrics and identify the best posting times, content types and platforms.

## Tools Used

* Python
* Pandas
* Power BI
* GitHub

## Analysis

* Loaded and explored the dataset
* Created Total_Engagement using Likes, Comments, Shares and Saves
* Analyzed engagement by hour
* Analyzed engagement by content type
* Analyzed engagement by platform

## Key Findings

* Best posting hours: 6 PM–7 PM
* Top content types: Duet, Stitch, Video
* Top platforms: TikTok, YouTube, Instagram

## Conclusion

Interactive content formats and evening posting times generated the highest engagement.
